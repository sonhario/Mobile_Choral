Dissimulation #V: Celular Coral

O trabalho é uma adaptação para web do trabalho Laptop Coral (Viena 2012). A versão original, concebida para a obra Bare Branches de Pia Palme para o festival Wien Modern, a qual inaugurou a série de trabalhos Simulacrum aqui é atualizada para um contexto tecnológico diferente, modulado por tecnologias ubíquas e disruptivas que dissimulam a formação de tecidos sônicos coletivos. Nesta versão o laptop é substituído por celulares que acessam o conteúdo disponível via web. A proposta é criar uma experiência coletiva, onde os participantes acessam conjuntamente o material sonoro via web. O trabalho reafirma o conceito original de coral maquínico através da coletividade em rede local, explora os dispositivos contemporâneos através de uma performatividade coletiva convidando os ouvintes/participantes a exercitar a escuta de um coro polifônico.

Ficha técnica Autor: Giuliano Obici
Ano: 2026
Dados técnicos: programação web
Voz: Salome Kammer interpretando solo de "Bare Branches" de Pia Palme
Desenvolvimento da versão web: Filipe Britto
